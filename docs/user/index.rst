User Guide
==========

Symfexit has two types of users: members, and admins (who are also members).
There are multiple ways in which you may be an admin, but all of these ways have in common that you can see the "Administration" menu button on the headers of the member area.
Clicking this will send you to the admin interface, where you can manage the Symfexit system.

You can also visit the admin area by going to `/admin/` on your Symfexit instance.

The usage of the admin interface is explained in the `admin section`_ of the user guide.
